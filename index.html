IntelliJ IDEA Setup
1
Open Project
  • Open IntelliJ IDEA
  • Select File â†’ Open... (or Open from Welcome screen)
  • Navigate to JavaCourse/java_project directory and click Open
  • IntelliJ should automatically recognize it as a Java project
2
Add JSON Library Dependency

The org.json library is required for JSON parsing operations. Choose one of the following methods:

Method A: Maven Configuration (Recommended)

If your project uses Maven, add this dependency to your pom.xml file:

<dependencies>
    <dependency>
        <groupId>org.json</groupId>
        <artifactId>json</artifactId>
        <version>20240303</version>
    </dependency>
</dependencies>

Maven Setup Steps:

  1. Create a pom.xml file in your java_project root directory
  2. Add the complete Maven project structure:
<?xml version="1.0" encoding="UTF-8"?>
<project xmlns="http://maven.apache.org/POM/4.0.0"
         xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
         xsi:schemaLocation="http://maven.apache.org/POM/4.0.0 
         http://maven.apache.org/xsd/maven-4.0.0.xsd">
    <modelVersion>4.0.0</modelVersion>
    
    <groupId>com.example</groupId>
    <artifactId>file-reader-app</artifactId>
    <version>1.0-SNAPSHOT</version>
    
    <properties>
        <maven.compiler.source>11</maven.compiler.source>
        <maven.compiler.target>11</maven.compiler.target>
        <project.build.sourceEncoding>UTF-8</project.build.sourceEncoding>
    </properties>
    
    <dependencies>
        <dependency>
            <groupId>org.json</groupId>
            <artifactId>json</artifactId>
            <version>20240303</version>
        </dependency>
    </dependencies>
    
    <build>
        <sourceDirectory>src</sourceDirectory>
        <plugins>
            <plugin>
                <groupId>org.apache.maven.plugins</groupId>
                <artifactId>maven-compiler-plugin</artifactId>
                <version>3.11.0</version>
            </plugin>
        </plugins>
    </build>
</project>
  1. In IntelliJ: File â†’ Open and select the java_project folder
  2. IntelliJ will detect the pom.xml and ask to import as Maven project - click Import
  3. Wait for Maven to download dependencies automatically
Method B: Manual JAR Download (Fallback)

If you prefer not to use Maven or encounter issues:

  • Download org.json JAR file from MVNRepository
  • Create lib folder in java_project directory
  • Place the downloaded JAR file in the lib folder
  • In IntelliJ: File â†’ Project Structure... (Ctrl+Alt+Shift+S)
  • Select Modules â†’ your project â†’ Dependencies tab
  • Click + button â†’ JARs or directories...
  • Select the JAR file and ensure scope is set to Compile
