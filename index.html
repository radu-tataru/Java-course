IntelliJ IDEA Setup
1
Open Project
  • Open IntelliJ IDEA
  • Select File → Open... (or Open from Welcome screen)
  • Navigate to JavaCourse/java_project directory and click Open
  • IntelliJ should automatically recognize it as a Java project
2
Add JSON Library Dependency

The org.json library is required for JSON parsing operations.

  • Download org.json JAR file from MVNRepository
  • Create lib folder in java_project directory
  • Place the downloaded JAR file in the lib folder
  • In IntelliJ: File → Project Structure... (Ctrl+Alt+Shift+S)
  • Select Modules → your project → Dependencies tab
  • Click + button → JARs or directories...
  • Select the JAR file and ensure scope is set to Compile
3
Run the Application
  • Navigate to Main.java in src/com/example/app package
  • Right-click on Main.java file
  • Select Run 'Main.main()' from context menu
Tip: You can also click the green play icon next to the main method in the editor.
