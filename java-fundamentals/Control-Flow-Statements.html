1. If-Else Statements

If-else statements allow your program to make decisions based on conditions.

Basic If Statement

int age = 18;

if (age >= 18) {
    System.out.println("You are eligible to vote");
}

If-Else Statement

int score = 75;

if (score >= 60) {
    System.out.println("Passed");
} else {
    System.out.println("Failed");
}

If-Else-If Ladder

int marks = 85;

if (marks >= 90) {
    System.out.println("Grade: A");
} else if (marks >= 80) {
    System.out.println("Grade: B");
} else if (marks >= 70) {
    System.out.println("Grade: C");
} else if (marks >= 60) {
    System.out.println("Grade: D");
} else {
    System.out.println("Grade: F");
}

Nested If Statements

int age = 25;
boolean hasLicense = true;

if (age >= 18) {
    if (hasLicense) {
        System.out.println("You can drive");
    } else {
        System.out.println("You need a license");
    }
} else {
    System.out.println("You are too young to drive");
}
Best Practice: Use curly braces {} even for single-line if statements to prevent bugs when adding more code later.

2. Switch Statements

Switch statements provide a cleaner way to handle multiple conditions based on a single variable.

Basic Switch Statement

int day = 3;
String dayName;

switch (day) {
    case 1:
        dayName = "Monday";
        break;
    case 2:
        dayName = "Tuesday";
        break;
    case 3:
        dayName = "Wednesday";
        break;
    case 4:
        dayName = "Thursday";
        break;
    case 5:
        dayName = "Friday";
        break;
    case 6:
        dayName = "Saturday";
        break;
    case 7:
        dayName = "Sunday";
        break;
    default:
        dayName = "Invalid day";
        break;
}

System.out.println("Day: " + dayName);  // Output: Day: Wednesday

Switch with Fall-Through

String month = "February";
int days;

switch (month) {
    case "January":
    case "March":
    case "May":
    case "July":
    case "August":
    case "October":
    case "December":
        days = 31;
        break;
    case "April":
    case "June":
    case "September":
    case "November":
        days = 30;
        break;
    case "February":
        days = 28;  // Not considering leap years
        break;
    default:
        days = 0;
        break;
}

System.out.println(month + " has " + days + " days");

Switch Expression (Java 14+)

int day = 3;

String dayName = switch (day) {
    case 1 -> "Monday";
    case 2 -> "Tuesday";
    case 3 -> "Wednesday";
    case 4 -> "Thursday";
    case 5 -> "Friday";
    case 6 -> "Saturday";
    case 7 -> "Sunday";
    default -> "Invalid day";
};

System.out.println("Day: " + dayName);
Warning: Always include a default case in switch statements to handle unexpected values. Don't forget break statements unless you intentionally want fall-through behavior.

3. For Loops

For loops are used when you know exactly how many times you want to execute a block of code.

Basic For Loop

// Print numbers 1 to 5
for (int i = 1; i <= 5; i++) {
    System.out.println(i);
}

// Output:
// 1
// 2
// 3
// 4
// 5

For Loop Structure

Part Description Example
Initialization Executed once at the start int i = 0
Condition Checked before each iteration i < 10
Update Executed after each iteration i++

Iterating Over Arrays

String[] fruits = {"Apple", "Banana", "Cherry", "Date"};

// Traditional for loop
for (int i = 0; i < fruits.length; i++) {
    System.out.println(i + ": " + fruits[i]);
}

// Enhanced for loop (for-each)
for (String fruit : fruits) {
    System.out.println(fruit);
}

Nested For Loops

// Print a multiplication table
for (int i = 1; i <= 5; i++) {
    for (int j = 1; j <= 5; j++) {
        System.out.print((i * j) + "\t");
    }
    System.out.println();
}

// Output:
// 1    2    3    4    5
// 2    4    6    8    10
// 3    6    9    12   15
// 4    8    12   16   20
// 5    10   15   20   25
Best Practice: Use the enhanced for-each loop when you need to iterate over all elements and don't need the index. It's more readable and less error-prone.

4. While Loops

While loops continue executing as long as a condition is true. Use them when you don't know the exact number of iterations in advance.

Basic While Loop

int count = 1;

while (count <= 5) {
    System.out.println("Count: " + count);
    count++;
}

// Output:
// Count: 1
// Count: 2
// Count: 3
// Count: 4
// Count: 5

Reading User Input

Scanner scanner = new Scanner(System.in);
String input = "";

while (!input.equals("quit")) {
    System.out.print("Enter a command (or 'quit' to exit): ");
    input = scanner.nextLine();
    System.out.println("You entered: " + input);
}

scanner.close();

File Reading Example

BufferedReader reader = new BufferedReader(new FileReader("data.txt"));
String line;

while ((line = reader.readLine()) != null) {
    System.out.println(line);
}

reader.close();
Infinite Loop Warning: Make sure your while loop has a condition that will eventually become false, or it will run infinitely!

5. Do-While Loops

Do-while loops execute the code block at least once before checking the condition.

Basic Do-While Loop

int number = 1;

do {
    System.out.println("Number: " + number);
    number++;
} while (number <= 5);

// Output:
// Number: 1
// Number: 2
// Number: 3
// Number: 4
// Number: 5

Menu-Driven Program

Scanner scanner = new Scanner(System.in);
int choice;

do {
    System.out.println("\n=== Main Menu ===");
    System.out.println("1. View Profile");
    System.out.println("2. Edit Settings");
    System.out.println("3. Logout");
    System.out.print("Enter your choice: ");
    choice = scanner.nextInt();

    switch (choice) {
        case 1:
            System.out.println("Viewing profile...");
            break;
        case 2:
            System.out.println("Editing settings...");
            break;
        case 3:
            System.out.println("Logging out...");
            break;
        default:
            System.out.println("Invalid choice!");
    }
} while (choice != 3);

scanner.close();

While vs Do-While

While Loop Do-While Loop
Condition checked before execution Condition checked after execution
May execute 0 or more times Executes at least once
while (condition) { ... } do { ... } while (condition);

6. Break and Continue

Break and continue statements control loop execution flow.

Break Statement

The break statement exits the loop immediately.

// Find the first number divisible by 7
for (int i = 1; i <= 100; i++) {
    if (i % 7 == 0) {
        System.out.println("First number divisible by 7: " + i);
        break;  // Exit the loop
    }
}

// Output: First number divisible by 7: 7

Continue Statement

The continue statement skips the rest of the current iteration and moves to the next one.

// Print odd numbers only
for (int i = 1; i <= 10; i++) {
    if (i % 2 == 0) {
        continue;  // Skip even numbers
    }
    System.out.println(i);
}

// Output: 1, 3, 5, 7, 9

Break with Nested Loops

// Break only exits the innermost loop
for (int i = 1; i <= 3; i++) {
    for (int j = 1; j <= 3; j++) {
        if (j == 2) {
            break;  // Breaks inner loop only
        }
        System.out.println("i=" + i + ", j=" + j);
    }
}

// Output:
// i=1, j=1
// i=2, j=1
// i=3, j=1

Labeled Break (Advanced)

// Break out of nested loops
outerLoop:
for (int i = 1; i <= 3; i++) {
    for (int j = 1; j <= 3; j++) {
        if (i == 2 && j == 2) {
            break outerLoop;  // Breaks outer loop
        }
        System.out.println("i=" + i + ", j=" + j);
    }
}

// Output:
// i=1, j=1
// i=1, j=2
// i=1, j=3
// i=2, j=1
Best Practice: Use break to exit loops early when you've found what you're looking for. Use continue to skip invalid or unwanted items.

7. Common Patterns and Examples

Pattern 1: Input Validation

Scanner scanner = new Scanner(System.in);
int age;

do {
    System.out.print("Enter your age (1-120): ");
    age = scanner.nextInt();

    if (age < 1 || age > 120) {
        System.out.println("Invalid age! Please try again.");
    }
} while (age < 1 || age > 120);

System.out.println("Valid age entered: " + age);
scanner.close();

Pattern 2: Searching in Arrays

String[] names = {"Alice", "Bob", "Charlie", "Diana"};
String searchName = "Charlie";
boolean found = false;

for (int i = 0; i < names.length; i++) {
    if (names[i].equals(searchName)) {
        System.out.println("Found " + searchName + " at index " + i);
        found = true;
        break;
    }
}

if (!found) {
    System.out.println(searchName + " not found");
}

Pattern 3: Sum and Average

int[] scores = {85, 90, 78, 92, 88};
int sum = 0;

for (int score : scores) {
    sum += score;
}

double average = (double) sum / scores.length;
System.out.println("Sum: " + sum);
System.out.println("Average: " + average);

Pattern 4: Finding Min and Max

int[] numbers = {45, 23, 67, 12, 89, 34};
int min = numbers[0];
int max = numbers[0];

for (int i = 1; i < numbers.length; i++) {
    if (numbers[i] < min) {
        min = numbers[i];
    }
    if (numbers[i] > max) {
        max = numbers[i];
    }
}

System.out.println("Minimum: " + min);  // 12
System.out.println("Maximum: " + max);  // 89

Key Takeaways

  • ✅ If-Else for Decisions: Use if-else statements for conditional logic and decision-making
  • ✅ Switch for Multiple Options: Use switch statements when comparing a single variable against multiple values
  • ✅ For Loops for Counting: Use for loops when you know the exact number of iterations needed
  • ✅ While Loops for Conditions: Use while loops when the number of iterations depends on a condition
  • ✅ Do-While for At Least Once: Use do-while loops when you need to execute code at least once
  • ✅ Break and Continue: Use break to exit loops early and continue to skip to the next iteration
  • ✅ Avoid Infinite Loops: Always ensure loop conditions will eventually become false
