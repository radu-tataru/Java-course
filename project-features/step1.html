First Attempt: The "Make It Work" Version

Learning Note: This is intentionally "messy" code that beginners might write. We'll improve it step by step in later lessons. The goal here is to understand the basics and see immediate results.

SimpleFileReader.java (Naive Version)

Let's start with a simple approach - everything in one main method:

package com.example.app;

import java.io.BufferedReader;
import java.io.FileReader;
import java.io.IOException;
import java.nio.file.Files;
import java.nio.file.Paths;
import org.json.JSONArray;
import org.json.JSONObject;

public class SimpleFileReader {
    public static void main(String[] args) {
        // Read TXT file - just read line by line
        System.out.println("=== Reading TXT File ===");
        try {
            BufferedReader txtReader = new BufferedReader(new FileReader("java_project/data/links.txt"));
            String line;
            while ((line = txtReader.readLine()) != null) {
                System.out.println("TXT Link: " + line);
            }
            txtReader.close();
        } catch (IOException e) {
            System.out.println("Error reading TXT file: " + e.getMessage());
        }

        // Read CSV file - split by comma
        System.out.println("\n=== Reading CSV File ===");
        try {
            BufferedReader csvReader = new BufferedReader(new FileReader("java_project/data/links.csv"));
            String line;
            while ((line = csvReader.readLine()) != null) {
                String[] parts = line.split(",");
                if (parts.length >= 2) {
                    System.out.println("CSV Link: " + parts[1].trim());
                }
            }
            csvReader.close();
        } catch (IOException e) {
            System.out.println("Error reading CSV file: " + e.getMessage());
        }

        // Read JSON file - use org.json library
        System.out.println("\n=== Reading JSON File ===");
        try {
            String jsonContent = new String(Files.readAllBytes(Paths.get("java_project/data/links.json")));
            JSONArray jsonArray = new JSONArray(jsonContent);
            for (int i = 0; i < jsonArray.length(); i++) {
                JSONObject obj = jsonArray.getJSONObject(i);
                System.out.println("JSON Link: " + obj.getString("url"));
            }
        } catch (Exception e) {
            System.out.println("Error reading JSON file: " + e.getMessage());
        }
        
        System.out.println("\nDone! All files processed.");
    }
}

Problems with This Approach

While this code works, notice the issues:

  • 🔄 Code Duplication: Similar file reading logic repeated three times
  • 🧱 Hard to Maintain: All code mixed together in one giant method
  • 🐛 Error Handling: Basic try-catch blocks don't provide much insight
  • 📁 Hard-coded Paths: File paths are written directly in the code
  • 🔧 No Flexibility: Adding a new file type means modifying the main method

But guess what? This is perfectly normal! Most developers start exactly like this. The key is recognizing these issues and improving the code iteratively.

Improved Implementation: Better Organization

From Messy to Better: Now let's refactor our naive code into something more organized. We'll extract methods to reduce duplication and make the code more readable.

SimpleFileReader.java

Create this file in java_project/src/com/example/app/ directory:


package com.example.app;

import java.io.BufferedReader;
import java.io.FileReader;
import java.io.IOException;
import java.nio.file.Files;
import java.nio.file.Paths;
import java.util.List;
import org.json.JSONArray;
import org.json.JSONObject;

public class SimpleFileReader {

    private static final String DATA_DIR = "java_project/data/";

    public static void main(String[] args) {
        System.out.println("--- Reading from TXT file ---");
        readTxtFile(DATA_DIR + "links.txt");

        System.out.println("\n--- Reading from CSV file ---");
        readCsvFile(DATA_DIR + "links.csv");

        System.out.println("\n--- Reading from JSON file ---");
        readJsonFile(DATA_DIR + "links.json");
    }

    private static void readTxtFile(String filePath) {
        try (BufferedReader reader = new BufferedReader(new FileReader(filePath))) {
            String line;
            while ((line = reader.readLine()) != null) {
                System.out.println("TXT Link: " + line);
            }
        } catch (IOException e) {
            System.err.println("Error reading TXT file: " + e.getMessage());
        }
    }

    private static void readCsvFile(String filePath) {
        try (BufferedReader reader = new BufferedReader(new FileReader(filePath))) {
            String line;
            while ((line = reader.readLine()) != null) {
                String[] parts = line.split(",");
                if (parts.length >= 2) {
                    System.out.println("CSV Link - Name: " + parts[0].trim() + ", URL: " + parts[1].trim());
                }
            }
        } catch (IOException e) {
            System.err.println("Error reading CSV file: " + e.getMessage());
        }
    }

    private static void readJsonFile(String filePath) {
        try {
            String content = new String(Files.readAllBytes(Paths.get(filePath)));
            JSONArray jsonArray = new JSONArray(content);
            for (int i = 0; i < jsonArray.length(); i++) {
                JSONObject obj = jsonArray.getJSONObject(i);
                System.out.println("JSON Link - Name: " + obj.getString("name") + ", URL: " + obj.getString("url"));
            }
        } catch (IOException e) {
            System.err.println("Error reading JSON file: " + e.getMessage());
        } catch (org.json.JSONException e) {
            System.err.println("Error parsing JSON file: " + e.getMessage());
        }
    }
}
Important: The org.json library is not part of the standard JDK. Add it as a dependency using the IntelliJ IDEA Setup Guide.
